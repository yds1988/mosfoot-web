# 🌐 MosFoot Web Interface
Статический веб-интерфейс для отображения статистики футбольного сообщества MosFoot

## 📁 Структура проекта
- `index.html` — главный файл сайта (с ключами Supabase)
- `config.js` — файл-помощник с настройками
- `README.md` — этот файл с инструкцией
- `404.html` — страница ошибки "404"

## 🚀 Запуск сайта
1. Откройте `index.html` двойным кликом
2. Сайт откроется в браузере
3. Данные загрузятся автоматически

## 📞 Поддержка
Если возникнут проблемы:
1. Проверьте консоль браузера (F12 → Console)
2. Убедитесь, что ключи правильные
3. Проверьте политики доступа в Supabase
